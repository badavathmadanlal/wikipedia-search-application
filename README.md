# Wikipedia Search Application

A responsive web application that allows users to search Wikipedia articles using the Wikipedia Search API. The application displays related search results with article titles, descriptions, and links. Clicking any result opens the corresponding Wikipedia page in a new browser tab.

---

## 📌 Project Overview

This application provides a simple and user-friendly interface to search Wikipedia articles instantly. It fetches live search results from the Wikipedia API and displays them dynamically without reloading the page.

---

## 🚀 Features

- Search Wikipedia articles in real-time
- Displays article title, description, and URL
- Opens selected article directly on Wikipedia
- Responsive user interface
- Fast search using Wikipedia API
- Clean and simple design

## ⚙️ How It Works

1. Enter a keyword in the search box.
2. Press **Enter** or click the **Search** button.
3. The application sends a request to the Wikipedia Search API.
4. Search results are displayed instantly.
5. Click any result to open the complete Wikipedia article.
